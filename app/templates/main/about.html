About This Application

Email Summarizer is an intelligent daily digest tool that helps you stay on top of your email and calendar without the overwhelm. Using advanced AI technology, it analyzes your Microsoft 365 emails and calendar to provide actionable insights.

Key Features

AI-Powered Analysis

Intelligent email classification using the 4D framework

Calendar Integration

Complete view of meetings with focus time calculation

Privacy First

Optional PII redaction to protect sensitive information

Healthy Habits

Once-daily digest to prevent email addiction

Technology Stack

Built with modern, reliable technologies:

  • Backend: Flask (Python)
  • AI: OpenAI GPT
  • Integration: Microsoft Graph API
  • Database: SQLAlchemy with SQLite/PostgreSQL
  • Frontend: Bootstrap 5

Open Source

This project is open source and available on GitHub. Contributions and feedback are welcome!
